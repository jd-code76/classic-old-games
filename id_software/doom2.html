Brief History of Doom II

Originally released on September 30th, 1994 for MS-DOS, Doom II built off the success of Doom and Ultimate Doom by adding a few new features, while keeping the graphics and technology largely the same. These features can be broken down as follows:

  1. New episode structure - 32-level "megaWAD" instead of three or four 9-level episodes. This change allows map makers to create continuous episodes where the player can keep his inventory for the entirety of the campaign.
  2. Added the super shotgun. Enough said.
  3. Added several new enemies - Revenant, Pain Elemental, Hell Knight, Chaingunner, Mancubus, Arch-vile, and Arachnotron (in addition to some secret enemies).

Doom II proved to be a great success for id and has since become the go-to version for classic Doom map makers and multiplayer enthusiasts. A 9-level expansion titled "No Rest for the Living" was developed by Nerve Software and released with the Xbox Live Arcade port of the game in 2010.

An updated version of the game titled "DOOM + DOOM II" was released on August 8th, 2024. This version, if you have a Bethesda.net account, allows you to download mods directly from the site, even on consoles.

Doom Source Ports

Chocolate Doom - Vanilla Doom experience that runs on Linux, Windows, and Mac.
Doom Retro - Vanilla Doom experience with a very minimalist approach. Runs on Windows.
Doomsday - Modern Doom experience with advanced features. Runs on Linux, Windows, and Mac.
ZDoom - Modern Doom experience that supports lots of idTech 1 games. Runs on Linux, Windows, and Mac.
ZDaemon - Doom multiplayer experience that is based on the ZDoom engine. Runs on Linux (partial), Windows, and Mac.
Zandronum - Doom multiplayer experience that is based on Skulltag. Runs on Linux, Windows, and Mac.
Doom source ports - A long list of source ports for Doom (the link is for Windows, but there are many other platforms listed).
